Detailed Description

This Header defines functions for registering the set of all variables on which maxsum::DiscreteFunction objects may depend.

Variables are uniquely identified by a key of type maxsum::VarID and each as a specified domain size. The purpose of the functions defined in this file are to register the domain size for each variable before it is used, and ensure that this size remains fixed throughout a programs execution. Variables can be registered multiple times, but in each case the domain size must not change. Variables must always be registered before they are referenced by any function.
